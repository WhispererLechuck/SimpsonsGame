Little game created by Pablo Rodriguez, which is a humble fan of this mythic show borned in the late 80's.

This webapp was created using basically Angular and Bootstrap. This idea appeared when I was trying to find ideas to practice Angular and then I found a funny REST API which provided quotes from the show.

Thanks to the developers who provided some material which (drastically) helped me in this task.
